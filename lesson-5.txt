2. HTML structure of a table
<table>
   <thead>          → Table heading section
         <tr>      → Table row
               <th> → Table header cell (bold text)
               <th> → Table header cell
 
   <tbody>          → Table body section
   <tr>      → Table row
   <td> → Table data cell
   <td> → Table data cell
   <tr>
   <td>
   <td>
</table>
